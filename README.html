2. RStudio and Package Installation

  • Make sure you can launch R studio successfully.

  • Try installing some packages

  • If this generates an error such as Tidyverse is not available for this R version, try it with another method:

3.2 Create Local Git Repository

  • The program we will use is called Git for Windows. You can right click anywhere on the desktop and choose Git Bash from the resulting dialogue box. This will open up a terminal session.
  1. First check where you are on the system

pwd
  1. Create a new folder

mkdir testgit

chdir testgit

pwd
  1. Initiate a new git repository in this location

git init
  1. Create some content


vim README.md
  1. This will open up a new file README.md in the vim text editor. Press the i key to start adding content to this file.

This is a test git repository.
  1. Press ESC key to exit the edit mode in the text editor. You are still inside the editor. Next, you will save and close the file.

  1. This will return you to your terminal session. Now letâ€™s work with git again.

  1. Add file to git

  1. Commit the changes to your repository

  1. Tell git about your GitHub account
